## Feature Description

Transform the level navigation UI from a flat circular layout to a hierarchical nested card grid structure:

- **Hierarchy**: Math Adventure → Operation (Addition/Subtraction/etc) → Levels
- **Layout**: Card grid layout instead of vertical circles
- **Display**: Level names shown instead of numbers
- **Visual Hierarchy**: Better visual distinction between hierarchy levels
- **Navigation Flow**: Improved UX for selecting operations and levels

  - Operation selection UI
  - Level display components
  - Routing/navigation logic
  - CSS/styling
